# Salary Prediction API 

A machine learning API built with Flask that predicts salary estimates based on user inputs such as education, experience, company size, and job title.

---

## Overview

This project was developed as part of a Business Analytics course project focused on machine learning model deployment and API development.

The API accepts JSON input and returns a predicted salary value using a trained machine learning model.

---

## Technologies Used

- Python
- Flask
- Joblib
- Machine Learning
- Docker
- Azure

---

## Features

### Salary Prediction
Predicts estimated salary values using a trained regression model.

### REST API
Provides API endpoints for external applications and testing tools.

### JSON Responses
Returns clean JSON outputs for integration with web applications.
